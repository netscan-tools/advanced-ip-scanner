<h1>Advanced IP Scanner — Free Network Scanner for Windows</h1>

<p><strong>Advanced IP Scanner</strong> is a free, fast, and portable network scanner for Windows. It scans local area networks (LAN) to detect all connected devices, retrieve IP addresses, MAC addresses, hostnames, and shared resources. Ideal for network administrators, IT technicians, and home users who need a reliable <strong>LAN IP scanner</strong> and <strong>network discovery tool</strong>.</p>

<h2>Key Features</h2>

<ul>
  <li><strong>Fast LAN scanning</strong> — discover all devices on your network within seconds across any specified IP range</li>
  <li><strong>Remote access to shared folders</strong> — browse shared resources including FTP, HTTP, and HTTPS services directly from scan results</li>
  <li><strong>Device identification</strong> — detect MAC address, hardware vendor, hostname, and operating system information for each device</li>
  <li><strong>Remote power management</strong> — send Wake-on-LAN, shutdown, and restart commands to networked PCs</li>
  <li><strong>Remote desktop integration</strong> — launch RDP or Radmin sessions to any discovered device</li>
  <li><strong>Portable mode</strong> — run without installation from a USB drive or any directory</li>
  <li><strong>Favorites list</strong> — save frequently accessed devices for quick targeted scans</li>
  <li><strong>Export results</strong> — save scan reports in CSV format for documentation and auditing</li>
</ul>

<h2>How It Works</h2>

<p>Advanced IP Scanner provides a clean, straightforward interface with a prominent scan control and result panels. You can fine-tune scan behavior — limit the address range, adjust scan speed, or filter results — all from the main window.</p>

<h3>Scan a Single IP Address</h3>

<ol>
  <li>Paste the target IP address into the entry field at the top of the window</li>
  <li>Click the <strong>Scan</strong> button</li>
  <li>Review returned information: MAC address, hostname, vendor, open ports, and response status</li>
</ol>

<h3>Scan Multiple IP Addresses</h3>

<ol>
  <li>Prepare a plaintext file with one IP address or IP range per line</li>
  <li>Import the file into Advanced IP Scanner</li>
  <li>Start the scan — the tool generates a detailed report for each entry</li>
</ol>

<h3>Use Favorites for Targeted Scans</h3>

<p>Mark specific devices as favorites to run targeted scans on a subset of machines instead of the entire network. This saves time when you only need to monitor specific endpoints or servers.</p>

<h2>Remote Control and Administration</h2>

<p>When paired with remote administration tools such as <strong>Radmin</strong> or Windows <strong>Remote Desktop Protocol (RDP)</strong>, Advanced IP Scanner enables:</p>

<ul>
  <li>Initiating remote desktop sessions directly from scan results</li>
  <li>Sending shutdown, restart, and Wake-on-LAN commands to networked computers</li>
  <li>Browsing shared folders, documents, images, and media on remote machines</li>
  <li>Accessing FTP and HTTP services discovered during the scan</li>
</ul>

<p>Because the program runs in portable mode, these capabilities are available from any Windows computer without installing the full suite.</p>
